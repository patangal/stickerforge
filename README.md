# StickerForge ✦

AI-powered sticker generator for Telegram & WhatsApp. Create custom stickers by describing what you want, picking a style, and downloading the result.

Built with [Pollinations.ai](https://pollinations.ai) — no server required.

## Features

- 🎨 **8 Style Presets** — Kawaii, Cartoon, 3D Render, Pixel Art, Minimalist, Watercolor, Retro, Emoji
- ⚡ **Instant Generation** — Describe anything and get a sticker in seconds
- 📥 **One-Click Download** — Save as PNG, ready for Telegram/WhatsApp
- 🔑 **Optional API Key** — Use free tier or bring your own Pollinations API key
- 🖼️ **Gallery** — Your recent stickers are saved locally
- 📱 **Fully Responsive** — Works on desktop, tablet, and mobile

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) → **New Project** → Import your repo
3. Click **Deploy** — no configuration needed
4. Done! Your sticker generator is live 🎉

## Tech Stack

- **HTML / CSS / JavaScript** — Zero dependencies, pure static site
- **Pollinations.ai** — Free AI image generation API
- **Vercel** — Static site hosting
